# 대성산업그룹 정적 웹사이트

첨부된 `대성산업그룹 홈페이지 내용.docx`의 정보구조와 원고를 바탕으로 만든 배포용 정적 웹사이트입니다.

## 구성

- `index.html` — 메인 페이지
- `assets/styles.css` — 전체 스타일
- `assets/script.js` — 배너 순환, 로그인 인증, 다크 모드 전환, 콘텐츠 렌더링

## 배포 방법

정적 호스팅 서비스에 이 폴더 전체를 업로드하면 됩니다.

- GitHub Pages: 저장소 루트에 업로드 후 Pages 활성화
- Netlify / Vercel: 새 프로젝트 생성 후 폴더 업로드
- 일반 웹호스팅: `public_html` 또는 웹 루트에 업로드

## 다크 모드 진입

우측 상단 `로그인` 버튼을 누르고 ISBN 마지막 4자리 `7624`를 입력하면 내부자료 모드로 전환됩니다.
승인 상태는 브라우저 `localStorage`에 저장됩니다. 화면의 `접근 해제` 버튼으로 초기화할 수 있습니다.

## 네이버 밴드 링크 교체

`assets/script.js` 파일의 다음 줄을 실제 초대장 URL로 교체하세요.

```js
const BAND_INVITE_URL = '#';
```
